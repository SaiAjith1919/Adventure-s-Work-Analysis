# 📊 Adventure Works Sales Analysis Dashboard

## 🚀 Project Overview

This project showcases an **interactive Business Intelligence dashboard** developed using **Power BI** to analyze the Adventure Works dataset.

The dashboard provides deep insights into **sales performance, customer behavior, product trends, and regional analysis**, enabling data-driven decision-making.

---

## 🎯 Key Features

* 📈 Dynamic Sales Trend Analysis
* 🛍️ Product Performance Insights
* 🌍 Region-wise Sales Distribution
* 👥 Customer Segmentation
* 💰 Profitability Analysis

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query (Data Transformation)
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)
* GitHub + Git LFS

---

## ⚙️ Techniques Used

### 🔹 Data Transformation (Power Query)

* Cleaned and structured raw data
* Removed null and duplicate values
* Created calculated columns
* Merged multiple datasets

---

### 🔹 Data Modeling

* Designed a **star schema**
* Built relationships between:

  * Sales (Fact Table)
  * Product, Customer, Date, Region (Dimension Tables)

---

### 🔹 DAX Calculations

Implemented advanced business metrics:

* Total Revenue
* Total Profit
* Profit Margin (%)
* Year-to-Date (YTD) Sales
* Month-over-Month Growth
* Customer Count
* Average Order Value

---
